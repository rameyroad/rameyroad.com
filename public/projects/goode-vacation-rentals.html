Goode Vacation Rentals

Goode Vacation Rentals is a vacation rental company in Hilton Head, SC. I designed, built, & host a custom website for displaying property information, searching availability, and taking online reservations.

The website integrates with a management REST API for property information & reservations.

Piranha CMS was integrated into the site to allow end users to manage all site content.

Goode Vacation Rentals was acquired by Vacasa in May 2020, and this site was decommissioned.

Technologies used:

  • Microsoft .Net Core 2.x, MVC, & jQuery
  • SCSS
  • Piranha CMS
  • Entity Framework Core & SQL Server
  • REST API integration
  • Azure DevOps build, deployment & web hosting
