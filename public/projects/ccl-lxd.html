Leadership By Design

Leadership by Design was a prototype personal leadership development app develped for leaders of all levels. With built-in learning campaigns, gamified content, and points tracking, LxD was a new way to approach leadership development that opened it to a larger audience of new and experienced leaders.

  • CHAT with ELLA about upcoming challenges. Practical and actionable ADVICE is at your fingertips.
  • Personalized assessments & learning journeys.
  • Rich leadership development campaigns

*LxD is currently on hold pending updated organization priorities.

Technologies used:

  • Microsoft .Net Core 6.x, .Net Web APIs
  • React Native & Expo GO
  • Entity Framework Core, Entity Framework, & Azure SQL Server
  • REST development & integration
  • Azure DevOps build, deployment & web hosting
